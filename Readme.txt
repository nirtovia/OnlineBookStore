כדי להתקין, תסים את קובץ ה-jar בכונן c, ואז תפעיל את cmd.
ב-cmd, תרשום את שורת הקוד הבאה:  java -jar {path}
{Path} ניתן לקבל בעזרת drag-and-drop של קובץ ה-jar לחלון ה-cmd.
לחץ enter.

כעת, המתן מספר רגעים כדי שהשרת יעלה, ואז תוכל להתחבר לאתר דרך localhost:8080

update: Since the jar file is too big, it cannot be uploaded. You would have to manually create a new JAR file or upload it to your platform. We used Intellij.
